Customer Segmentation & Churn Pattern Analytics in European Banking

Project Description:
This project studies customer churn in the European banking sector.
Customer churn refers to customers closing or discontinuing their bank accounts.
The project uses customer segmentation to identify high-risk customer groups.

Objectives:
- To analyze customer churn patterns
- To segment customers based on demographic and financial factors
- To understand factors influencing customer retention

Dataset:
The project uses secondary banking data containing customer demographic,
financial, and behavioral variables such as age, gender, geography, balance,
tenure, credit score, and churn status.

Tools Used:
- MS Excel
- MS Word

Conclusion:
The analysis helps banks understand churn behavior and design better
customer retention strategies.
